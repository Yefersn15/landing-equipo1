# Landing Equipo 1

Este proyecto es una landing page inspirada en la tienda oficial de LEGO, desarrollada como trabajo grupal.

## Descripción

La página presenta una interfaz visualmente atractiva con secciones de navegación, promoción de productos destacados, contenido especial y un diseño moderno parecido al estilo de la marca.

## Estructura del proyecto

- index.html: estructura principal de la landing page
- CSS/: archivos de estilos
- IMG/: imágenes utilizadas en la página
- JS/: scripts del proyecto
- ProductosDestacados/: recursos relacionados con productos destacados

Abre el archivo index.html en tu navegador para visualizar la landing page.
